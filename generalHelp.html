General Help

A data file can be in one of several formats: Tab Separated Value, Comma Separated value or a standard Excel file. TSV & CSV files contain lines separated by the NewLine character ('\n'). Lines starting with the '#' character are ignored as comments. The first non-comment line (or row) contains names of the recorded Variables. Subsequent lines contain values for the variable in that row. The first value in each line should be the time.

You can open a different file via the File menu item. This viewer does not modify files.

A list of all available variables is displayed. Clicking on a variable selects it. You can click on multiple variables. Clicking on a selected variable un-selects it. Each variable is displayed in a different color. The color of the plot corresponds to the color of the variable.

Click on a column header to sort (or reverse sort) by that column

Help with plots

Each variable plotted has its own axis. This may appear to the right or left

Use the mouse wheel to zoom in and out of a point in the frame.

Left click on the plot area and move the mouse to pan.

When multiple variables are being displayed, move the mouse to a specific axis and drag it up or down. This will move the associated variable.

Roll the mouse wheel over the axis to change the Y scale. Alternatively, right click on the plot area and move the mouse up or down.

Displaying Log Y will fail if any displayed variable has a 0 value.

Reset Plot to re-center all plots within the frame.
