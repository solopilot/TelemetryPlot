Help with plots

Each variable plotted has its own Y axis with scale. Additional axes may appear on the left. Each is identified by the colored variable name.

Use the mouse wheel to zoom in and out of a point in the frame. The zoom centers on the mouse position.

Left click on the plot area and move the mouse left or right to pan along the X axis (time).

Left click on the plot area and move the mouse up or down to pan the main plot which has its axis on the left. To move other plots, left click on the associated Y axis and move the mouse up or down. Jiggling it helps locate a plot among all other plots.

Right click on the plot area and move the mouse horizontally to change the scale of the X axis for all plots. Right click and move vertically to change the scale for the main plot which has its axis on the left. To scale other plots, move the mouse over the associated Y axis and move the mouse up or down, or use the scroll wheel.

Click "Reset Plot" to re-center all plots within the frame. This is useful when you are done playing with mouse buttons and have messed up the plots.

Click "Compress Y Axis" to make the Y Axis a Log scale. This is actually not terribly helpful since most data is linear. Click it again after you discover that I am right.

General Help

A list of all available variables is displayed. Clicking on a variable selects it. You can click on multiple variables. Clicking on a selected variable un-selects it. Each variable is displayed in a different color. The color of the plot corresponds to the color of the variable.

Click on a column header to sort (or reverse sort) by that column

Variables that do not change value in the recording are called "inactive". They are marked in red. Plotting them will result in a boring flat line. You can hide them by clicking on the "Hide" checkbox.

You can open a different file via the File menu item. This viewer does not modify, mutilate, harrass or vilify files as yet.

A data file can be in one of several formats: Tab Separated Value, Comma Separated Value or a standard Excel file. TSV & CSV files contain lines separated by the NewLine character ('\n'). Lines starting with the '#' character are ignored as comments. The first non-comment line (row) should contain names of the recorded Variables. Subsequent lines contain values for the variable in that row. The first value in each line should be the time. Variable values can be numeric or text. If text, then the range of values should be limited, like [ Initial, Looking for Target, Target Acquired, Shooting, Target Destroyed, Missed - Dammit, Done ]
